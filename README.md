코딩 프로젝트 A–Z는 알파벳 A부터 Z까지 각 글자에 대응하는 총 26개의 프로그램을 직접 설계하고 구현하는 개인 개발 프로젝트입니다.
각 프로젝트는 C, Java, Python 등 서로 다른 프로그래밍 언어를 활용하여 다양한 주제의 기능을 구현하며, 언어별 특징과 장점을 비교·체험할 수 있도록 구성되어 있습니다.

본 프로젝트는 단순히 코드를 작성하는 데서 끝나는 것이 아니라, "하나의 기능을 완성된 프로그램 형태로 만들어 내는 경험" 자체를 목표로 하고 있습니다.
따라서 각 항목은 복잡한 대규모 시스템이 아닌, 명확한 목적을 가진 소규모 실용 프로그램 또는 실습용 도구들로 구성되어 있으며, 
이를 통해 핵심 개념을 체계적으로 익힐 수 있었습니다.

각 프로그램에서는 다음과 같은 다양한 기술들을 자연스럽게 사용하게 됩니다:

- 기초 문법 및 언어별 특성 이해
변수, 함수, 클래스 구조, 예외 처리 등 기본 요소를 직접 활용

- 자료구조와 알고리즘 활용
배열, 리스트, 스택, 큐, 정렬 및 탐색 알고리즘 구현 등

파일 입출력 및 데이터 처리 경험
CSV, TXT, 로그 파일 등 다양한 형식의 데이터를 읽고 쓰는 실습

간단한 게임 로직과 인터랙션 구현
텍스트 기반 게임, 난이도 조절, 점수 기록 등 로직 중심의 구조 설계

API 활용 및 시각화 경험(Python)
간단한 웹데이터 수집, 그래프 시각화 등 실용적인 기능 학습

프로그래밍 사고 확장
기능 분할, 함수 재사용, 모듈화 등 실제 개발 프로세스 기반의 설계 경험

각 프로그램은 독립적으로 동작하도록 제작되어 있어, 하나의 프로젝트를 완성할 때마다 개발 과정 전체를 경험할 수 있으며, 이를 통해 작은 성공 경험을 여러 번 축적하는 구조로 되어 있습니다.
또한, 동일한 문제를 다른 언어로 구현해볼 수 있기 때문에, 언어 간 차이점과 특징을 쉽게 비교할 수 있는 점도 본 프로젝트의 장점입니다.

마지막으로, 프로젝트 A–Z는 포트폴리오로도 활용하기 쉽도록 정리되어 있으며, GitHub를 통해 열람할 수 있도록 제작되었습니다. 
이를 통해 개발자로서의 성장 과정과 학습 결과를 한눈에 확인할 수 있습니다.
